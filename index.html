Internet & Web Basics

The Internet is a worldwide telecommunications system that provides connectivity for
millions of other, smaller networks. The Internet is referred to as a network of networks.
It allows computer users to communicate with each other across distance and computer
platforms. The protocol used for web resources is HyperText Transfer Protocol (HTTP).

Cascading Style Sheet (CSS3)

CSS, on the other hand, adds styling in form of colors, shapes, fonts and other design elements
while JavaScript can be used to make things more interactive and influence how elements behave.
CSS is stands for Cascading Style Sheets, it is the primary language used to describe look and
formatting for webpages across the HTML and XML in general, for us we used it along with HTML5.
CSS allows us to easily apply rules about formatting and layout to our HTML elements and then re-use
those rules across multiple elements and even pages. I learned some CSS codes and how to use. Among
those coding are; syntax for color values, configuration of Inline, Embedded, and External CSS, CSS
selectors such as Id, class, and descendant. Span element, the cascade, CSS syntax validation, how
to edit width, height, margin, borders, and padding with CSS

Graphics and Text Styling Basics

CSS graphics and text styling are a shorter and sweeter method of controlling font display, and are
supported by modern browsers. I learned how they work, how to set up and position images, how to

Client doesn't like the Arial? Wants a new text color scheme? No problem! A quick edit, and you're done.
